# SM20 Checklist (v1) — Diretrizes de Qualidade (NÃO é conteúdo do usuário)

Regras abaixo são APENAS para FORMATAÇÃO/QUALIDADE dos cards.
NUNCA crie cards sobre estas regras.

[SM01 ENTENDER] Se não dá pra explicar com suas palavras, NÃO gere card.
[SM02 VISÃO GERAL] Comece pelo essencial: definição + “pra que serve” antes de detalhes.
[SM03 BÁSICOS] Priorize fundamentos estáveis; eles destravam o resto.
[SM04 ATOMICIDADE] 1 card = 1 fato/1 distinção/1 definição. Se tiver “e/ou/lista”, DIVIDA.
[SM05 CLOZE] Cloze deve lacunar só 1 termo-chave; resto vira pista. Evite cloze “cego”.
[SM06 IMAGÉTICO] Se o conteúdo for visual, use pistas visuais/textuais curtas (sem inventar).
[SM07 MNEMÔNICO] Só quando necessário: use uma dica curta (não escreva uma história).
[SM08 OCULTAÇÃO] Para diagramas/partes, prefira “nome da parte X” (se o texto suportar).
[SM09 SEM SETS] Evite “liste N itens”. Transforme em vários cards atômicos.
[SM10 SEM ENUM] Se for sequência, use clozes sobrepostos ou quebrar em blocos pequenos.
[SM11 INTERFERÊNCIA] Diferencie similares explicitamente (contraste X vs Y).
[SM12 REDAÇÃO] Pergunta curta e específica; resposta ainda mais curta (10–25 palavras).
[SM13 ASSOCIAÇÕES] Se ajuda a desambiguar, use contexto mínimo (“no contexto de ___”).
[SM14 EXEMPLO] Use exemplo do próprio texto quando ele reduzir ambiguidade.
[SM15 EMOÇÃO] Não force. Só use se o texto já trouxer um gancho marcante.
[SM16 PISTAS] Use rótulos curtos (“bio:”, “math:”) para evitar confusão entre domínios.
[SM17 REDUNDÂNCIA BOA] Pode haver 2 cards do mesmo conceito se cada um treina algo diferente.
[SM18 FONTE] Sempre inclua SRC literal (5–25 palavras) que comprove o card.
[SM19 DATA/VERSÃO] Se houver números/versões/ano no texto, preserve como contexto (sem inventar).
[SM20 PRIORIZE] Gere mais cards para conceitos centrais; menos para detalhes periféricos.

GATES (checagens rápidas):
- Se a resposta tem 2+ ideias → dividir.
- Se o card parece genérico (“o que é X?” sem contexto) → especificar.
- Se não existe SRC literal no texto → descartar.
- Se virou lista grande → quebrar em itens menores.
